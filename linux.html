Programs for Linux distributions

Basic required programs for various distributions on the Linux kernel

Programs taken from official sources

Google Chrome

Google Chrome is a browser developed by Google based on the free Chromium browser and the Blink engine.

Mozila FireFox

Mozilla Firefox is a free Quantum browser developed and distributed by Mozilla Corporation.

Chromium

Chromium is an open source web browser developed by The Chromium Authors, Google and several other companies.

Wine

Free software that allows users of UNIX-like operating systems, such as GNU/Linux, Apple macOS, and other UNIX-like operating systems with support for x86, AMD64 CPU architectures (and other CPU architectures, if there is compatibility of launched applications with them) to execute 16-, 32- and 64-bit applications, including games created exclusively for Microsoft Windows without having Microsoft Windows installed

Open Office

Free package of office applications. Competed with commercial office suites both at the level of formats and at the level of the user interface. One of the first to support the new open format OpenDocument

VLC

A free cross-platform media player developed by the VideoLAN project

GIMP

A freely distributed raster graphics editor, a program for creating and processing raster graphics with partial support for working with vector graphics.
