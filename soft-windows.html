Programs for Windows

Here are collected programs for PC based on Windows OS! See if you can find something useful!

Windows images can be found here (Source not mine!)

Google Chrome

Chrome is a browser developed by Google based on the free Chromium browser and the Blink engine.

Mozila FireFox

Mozilla Firefox is a free Quantum browser developed and distributed by Mozilla Corporation.

Chromium

Chromium is an open source web browser developed by The Chromium Authors, Google and several other companies.

Process Hacker

A free open source utility for monitoring system processes and services running under 32-bit and 64-bit operating systems of the Windows family, used as a replacement or addition to the Windows Task Manager.

7-Zip

Free file archiver with a high degree of data compression. Supports multiple compression algorithms and many data formats, including native 7z format with highly efficient LZMA compression algorithm.

Mem Reduct

Lightweight real-time memory management application to monitor and clean system memory on your computer.

CPU-Z

A free utility application for displaying technical information about a user's personal computer, running under Microsoft Windows starting from Windows 95. A special version for Android is also available.

Winaero Tweaker

A free application for all versions of Windows that allows you to configure (i.e. configure) hidden secret settings that Microsoft does not allow you to configure from the user interface.

Victoria

A computer program designed to assess the health, testing and minor repairs of hard drives. It was developed by an individual entrepreneur from Moscow, who now lives and works in Minsk, and an information recovery specialist Sergey Olegovich Kazansky. Free.

AIDA64

FinalWire Ltd. utility. for testing and identifying components of a personal computer running Windows operating systems, providing detailed information about hardware and software.

Visual Studio Code

Source code editor developed by Microsoft for Windows, Linux and macOS. It is positioned as a "light" code editor for cross-platform development of Web and cloud applications.

Mi Unlock Tool

Xiaomi Bootloader Unlock Tool (Only working version)

ADB & Fastboot Drivers

Drivers for devices based on Android for setting up, restoring your phone via PC

Odin

The official utility for PC that allows you to update the firmware or install the modem, kernel or other firmware components of your smartphone or tablet from Samsung on the Android platform. Attention!!! Odin is only compatible with Samsung Android devices!

Samsung USB Driver

USB drivers exclusively for Samsung devices

Scrcpy

This app provides display and control of Android devices connected via USB or TCP/IP. It does not require any ROOT access. It works on GNU/Linux, Windows and MacOS

PowerToys

Microsoft PowerToys is a set of freeware system utilities designed for power users developed by Microsoft for use on the Windows operating system.

SpaceDesk

A program for connecting additional monitors to a computer via a network, which can be used as other computers (for example, laptops) or video walls with mobile devices
