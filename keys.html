Activating Windows Without Activators Using CMD (Command Prompt)

1. To get started, open a command prompt as an administrator. To do this, in the search, write "Command Prompt", press RMB and select "Run as administrator"

2. After opening enter: slmgr /ipk "selected-key"

2-1. A window about installing the key will appear.

3. We enter the following command: slmgr /skms kms.digiboy.ir

3-1. We get a window about connecting to the server:

4. Enter the last command: slmgr /ato

4-1. We get a window about successful activation

If it fails, try these command in the second step: slmgr /skms zh.us.to or try changing the license key
